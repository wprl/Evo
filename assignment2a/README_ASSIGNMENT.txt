The only external thing needed to build this is ant, which is available from apt-get.

To build: 

ant jar

To run:

cd dist
java -jar assignment2a.jar [--config <configfile>]

default.cfg		   configuration for experiment
combo_n_log.txt		   log for experiment 
combo_n_solution.txt	   solution for experiment 
chart.png		   graph of fitness vs. evals
