The only external thing needed to build this is ant, which is available from apt-get.

To build: 

ant jar

To run:

cd dist
java -jar assignment1a.jar [--config <configfile>]

combo_n.cfg		   configuration for experiment with combo n
combo_n_log.txt		   log for experiment n
combo_n_solution.txt	   solution for experiment n
combo_n_chart.pdf	   chart of mean mean fitness for combo n
analysis.ods		   LibreOffice spreadsheet of statistical analysis (to show work)
analysis.pdf		   PDF export of above file (for compatibility)
